# Descrição do projeto
Aplicativo exemplo para demonstrar testes unitários.

Este aplicativo fornece o resultado do Cálculo de IMC de acordo com os níveis:
- Magreza Severa
- Magreza moderada
- Magreza leve
- Saudável
- Sobrepeso
- Obesidade Grau I
- Obesidade Grau II (severa)
- Obesidade Grau III (mórbida)
